About RaceSim Analyzer

QA Regression Testing for Racing Simulations. It is just a POC/DEMO

What is RaceSim Analyzer?

RaceSim Analyzer is a QA regression testing tool for racing car simulations. It compares Monte Carlo simulation results between car versions and provides AI-powered insights to help engineers make data-driven decisions.

The tool analyzes candidate configurations against baselines across key metrics such as lap time, fuel consumption, tire degradation, and brake temperatures, flagging regressions and highlighting improvements.

Key Features
  • Statistical comparison of candidate vs baseline runs (mean, std, delta)
  • YAML-based requirements validation with weighted scoring
  • Position-based telemetry analysis with threshold crossing detection
  • Visualization: dashboards, delta plots, and violation maps
  • Comprehensive PDF report generation
  • AI analysis via Ollama (local) or OpenAI API (GPT-4o)
  • Interactive LLM chat with database tool calling
  • MCP server for external LLM clients (Claude Desktop, etc.)
